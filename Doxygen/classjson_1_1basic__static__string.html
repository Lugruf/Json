Detailed Description

template<typename CharT, typename Traits = std::char_traits<CharT>>
class json::basic_static_string< CharT, Traits >

Representation of a constant string that will outlive the json::value that is using it (usually a string literal). A json::value can use static strings as keys for objects in order to avoid allocations if the user knows for sure that the string will not be destroyed while the value is being used.
